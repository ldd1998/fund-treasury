## FundTreasury

### What is this?
一个去中心化资金金库合约，用于管理 ETH 的存入、权限控制与提取。

### Key Features
- anyone can deposit ETH
- only owner can withdraw
- all contracts verified on Sepolia

### Tech Stack
- Solidity
- Hardhat
- hardhat-deploy
- Sepolia
- Etherscan verified

### Deployment
npx hardhat deploy --network sepolia
